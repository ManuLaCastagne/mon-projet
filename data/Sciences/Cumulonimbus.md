###### Questions

Quel type de nuage, capable de s'étendre verticalement sur 12 000 mètres, est associé aux orages les plus puissants dans l'atmosphère terrestre ?

Quel type de nuage peut provoquer de violentes turbulences dangereuses pour l'aviation, selon les études météorologiques menées par [[l'Organisation Météorologique Mondiale]] depuis 1950 ?

Quel type de nuage, souvent observé dans les zones tropicales, est responsable du développement de phénomènes extrêmes tels que les tornades, suite aux recherches publiées par [[la NASA]] en 2012 ?

###### Description

Le [[Cumulonimbus]] est un type de nuage dense et vertical, souvent associé aux conditions météorologiques extrêmes, comme les orages et les tempêtes, et il est fréquemment observé dans des villes comme [[Tampa]], célèbre pour ses orages. 

Ces nuages peuvent atteindre des hauteurs impressionnantes, parfois jusqu'à 20 kilomètres, et leurs sommets peuvent s'étendre dans la stratosphère. 

Un fait étonnant est que le [[Cumulonimbus]] détient le record pour le nuage ayant produit les grêlons les plus lourds jamais enregistrés, pesant jusqu'à 1,02 kg, en [[Bangladesh]] en 1986. 

Ces nuages sont aussi responsables de la formation de phénomènes spectaculaires tels que les éclairs intra-nuageux, rendant le [[Cumulonimbus]] non seulement fascinant pour les météorologues mais aussi une source d'inspiration pour les amateurs de climatologie.
