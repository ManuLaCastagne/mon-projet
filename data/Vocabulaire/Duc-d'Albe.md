###### Questions

Quel est le nom de l'ouvrage d'amarrage utilisé couramment dans les ports pour sécuriser les navires, qui porte le nom d'un célèbre personnage espagnol du XVIe siècle ?

Comment appelle-t-on la structure maritime érigée dans certaines baies et rades pour fournir un point d'attache aux embarcations, et qui partage son nom avec un général et homme politique des [[Pays-Bas espagnols]] ?

Quel terme est utilisé pour désigner un type de pieu implanté dans l'eau pour maintenir à quai les bateaux, nommée d'après le [[duc d'Albe de Tormes]], représentant de [[Philippe II]] durant l'occupation espagnole des [[Pays-Bas]] ?

###### Description

Le terme "Duc-d'Albe" fait référence initialement à [[Fernando Álvarez de Tolède]], le troisième [[duc d'Albe]], un général espagnol du XVIe siècle connu pour ses campagnes militaires.

Dans le domaine maritime, un "duc-d'Albe" désigne une structure constituée de pieux enfoncés dans le lit d'un plan d'eau, utilisée pour accoster ou amarrer des navires, un terme qui aurait trouvé son origine dans les ports de [[Flandre]].

Cette installation est souvent visible dans les ports d'Europe, notamment en [[France]], car elle offre une solution stable pour immobiliser les bateaux sans nécessiter de quai en dur.

Ancré dans la tradition et l'histoire, le rôle du "duc-d'Albe" s'avère encore essentiel dans les infrastructures portuaires modernes, témoignant ainsi de l'évolution des techniques maritimes depuis l'époque de [[Fernando Álvarez de Tolède]].
