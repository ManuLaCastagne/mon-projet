###### Questions

Quel est le terme médical utilisé pour désigner l'ongle chez l'être humain ?
###### Description

Le terme "Unguis" provient du latin, et désigne généralement les structures cornées, comme les ongles ou les griffes, chez les mammifères. 

Dans l'Antiquité, les [[Romains]] considéraient les soins des ongles comme un signe important de statut social et de raffinement. Le célèbre botaniste et médecin [[Carl von Linné]], également connu sous le nom de [[Linneaus]], utilise le mot "unguis" dans la classification des espèces pour décrire certaines caractéristiques physiques chez les animaux. 

Aujourd'hui, dans le domaine de la zoologie, "unguis" est encore employé pour désigner les plaques cornées des pattes d'insectes ou autres arthropodes, illustrant ainsi l'évolution des usages du mot à travers les siècles.
