###### Questions

Quel est le nom de ce petit [[compas]] dont l'une des branches est tranchante ?

###### Description

La "Rouannette" est une petite rouanne, un compas dont l’une des branches est tranchante et qu’employaient les agents des contributions indirectes pour marquer les pièces de vin, les charpentiers pour marquer les bois de charpente.
