###### Questions

Quel est le terme latin utilisé pour désigner un représentant légal appointé pour défendre les intérêts d'une personne lors d'une procédure judiciaire, notamment lorsque celle-ci est mineure ou incapable de se représenter elle-même ?

Comment appelle-t-on une personne nommée par un tribunal pour représenter une personne spécifique dans le cadre d'une affaire judiciaire spécifique, souvent utilisé dans le contexte de la [[justice]] [[familiale]] ou [[successorale]] ?

Quel est le terme qui désigne, dans le langage juridique anglo-saxon, un avocat ou un représentant désigné pour agir pour le compte d'une partie lors d'un litige et qui se limite à cette affaire particulière ?

###### Description

Le terme "Ad Litem" est une expression latine utilisée principalement dans le domaine juridique pour désigner une personne nommée par un tribunal afin de représenter les intérêts d'une partie, souvent mineure ou inapte, dans une affaire légale. Par exemple, dans la célèbre affaire de [[Brown v. Board of Education]] aux États-Unis, un curateur ad litem aurait pu être nommé pour représenter les enfants impliqués si nécessaire. En France, cet outil est également usité, notamment dans des cas de successions litigieuses ou de tutelle des mineurs devant les tribunaux de [[Paris]]. Historiquement, le principe d'ad litem souligne l'importance accordée par le droit de garantir que toutes les parties, même les plus vulnérables, aient une voix représentée équitablement dans les procédures judiciaires.
