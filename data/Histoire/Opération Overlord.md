###### Description

L'[[Opération Overlord]] est le nom de code désignant le débarquement allié en [[Normandie]], qui a commencé le [[6 juin 1944]], marquant le début de la libération de l'Europe occidentale occupée par l'Allemagne nazie pendant la [[Seconde Guerre mondiale]].

Sous le commandement du général [[Dwight D. Eisenhower]], cette opération militaire fut la plus grande invasion amphibie de l'histoire, mobilisant plus de 156 000 soldats des nations alliées, notamment les États-Unis, le Royaume-Uni et le Canada.

Un fait étonnant est que l'opération a été planifiée dans le plus grand secret, et un système de désinformation a été mis en place, connu sous le nom d'[[Opération Fortitude]], pour tromper les forces allemandes sur le lieu du débarquement, les poussant à croire que la vraie invasion aurait lieu dans le [[Pas-de-Calais]].

Nombreux sont ceux qui ignorent que l'Opération Overlord a nécessité la construction de deux ports artificiels, appelés [[Mulberry harbours]], qui ont été installés dans les jours suivant le débarquement pour faciliter l'approvisionnement des forces alliées sur les plages normandes.
